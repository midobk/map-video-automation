# Map fixtures

## Neutral fixture

`map-video-neutral.ts` is a deterministic, public-domain plan that exercises:

- title, map-highlight, comparison, caption, outro scenes
- English captions
- a deterministic vector world map rendered with D3 Geo / Natural Earth

The map scene labels the six continents and uses the `natural-earth` projection.
No static map asset is required.

## RTL fixture

`map-video-rtl.ts` is the same plan translated into Arabic with `direction: rtl`,
covering line-splitting, layout mirroring, and Arabic text on vector map labels.

## Country-zoom fixture

`map-video-country-zoom.ts` zooms the vector map to Morocco, Canada, Algeria, and
France. Canada and France are multipart / island-rich, which verifies that the
D3 Geo fit handles multi-polygon features. Each country scene is a separate
`map-highlight` with `focusIsoCodes` set to the ISO3 code.

## Ranking fixture

`map-video-ranking.ts` ranks continents by land area. It exercises the `ranking`
scene and a `stat-card` scene that displays a single large metric.

## Geographic data

Vector geography comes from `world-atlas` (Natural Earth, public domain) via
`d3-geo` and `topojson-client`. Country names are resolved through a generated
ISO3 dictionary derived from public-domain `world-countries`. Provenance and a
full data manifest are generated by `scripts/prepare-geo-data.ts` into
`src/geo/dataset-manifest.ts`.
